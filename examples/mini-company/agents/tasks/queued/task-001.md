## What to Do

Write a landing page for TaskFlow — a project management SaaS for freelancers. The page should:

1. **Explain what TaskFlow does** in one sentence a freelancer would understand
2. **Highlight three key features**: project tracking, time logging, and simple invoicing
3. **Include an email signup** for the beta waitlist (just a form — no backend needed yet)
4. **Look professional** but simple. No framework — plain HTML and CSS. Mobile-friendly.

Save the landing page to `products/taskflow/code/index.html`.

## Who This Is For

Freelancers who juggle multiple clients. They're tired of using three different tools for projects, time, and invoices. TaskFlow brings it together in one place.

## What "Done" Looks Like

- A single HTML file at `products/taskflow/code/index.html`
- Opens in a browser and looks professional
- Clearly communicates what TaskFlow is
- Has a working email signup form (can be a static form that doesn't submit anywhere yet)
- No placeholder text — every word is intentional
- Mobile-responsive

## Context

This is the first thing anyone will see from TaskFlow. It's also the first thing this company has ever produced. Make it count.
